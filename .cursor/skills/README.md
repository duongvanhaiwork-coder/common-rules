# Shared Skills

Core reusable skills for all projects (11 skills):

- `shared-planning-execution`
- `shared-code-quality`
- `shared-systematic-debugging`
- `shared-git-workflow`
- `shared-auto-commit-assistant`
- `shared-verification-gate`
- `shared-security-baseline`
- `shared-communication-handoff`
- `shared-docs-change-notes`
- `shared-operational-readiness`
- `shared-dependency-config-hygiene`

## Suggested Order Of Use

1. `shared-planning-execution`
2. `shared-systematic-debugging` (if issue-driven)
3. `shared-code-quality`
4. `shared-security-baseline`
5. `shared-verification-gate`
6. `shared-communication-handoff`
7. `shared-git-workflow`
8. `shared-auto-commit-assistant` (when you want staged + message-ready or authorized auto-commit)
9. `shared-docs-change-notes` (when behavior/contract changed)
10. `shared-dependency-config-hygiene` (when dependency/config changes)
11. `shared-operational-readiness` (before merge/release)
