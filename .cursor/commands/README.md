# Shared Cursor Commands

This directory stores reusable command prompts for all projects.

## Suggested Usage

- Copy selected files to a target repository at `.cursor/commands/`.
- Keep command names stable and descriptive.
- Keep backend/frontend specific commands in separate files when needed.

## Current Commands

- `plan-task.md`: Create an implementation plan before coding.
- `implement-task.md`: Implement with verification and concise reporting.
- `review-changes.md`: Review changes with bug/risk-first mindset.
- `ship-checklist.md`: Final pre-merge checklist for quality and release safety.
- `ops-readiness-check.md`: Validate operational readiness and rollback safety.
